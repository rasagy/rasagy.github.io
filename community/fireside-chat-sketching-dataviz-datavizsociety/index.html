I got a chance to join two incredible designers — Katherine Haugh & Duncan Geere — for a fireside chat on Sketching & Data Visualization. This chat was done virtually, moderated by Amy Cesal for the DataVizSociety.

We spoke about using sketching in the data viz process, using sketching as an aesthetic for data viz, and using sketchnoting as a way to visually represent information.
